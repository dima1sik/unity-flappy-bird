# Flappy Bird Unity Clone

Flappy Bird Unity Clone is a 2D Unity game prototype inspired by the classic Flappy Bird gameplay loop.

## Overview

The project demonstrates simple arcade-game mechanics in Unity: player movement, obstacle spawning, collision handling, scoring, and 2D scene setup.

## Features

- 2D player movement
- Physics-based gameplay
- Obstacle spawning
- Collision detection
- Score tracking
- Unity prefab-based structure
- Simple arcade gameplay loop

## Tech Stack

- Unity
- C#
- 2D Game Development
- Unity Physics
- Unity Prefabs

## Project Structure

- Assets/
- Packages/
- ProjectSettings/

## How to Open

1. Clone this repository.
2. Open the project folder in Unity Hub.
3. Use the Unity version listed in `ProjectSettings/ProjectVersion.txt` if possible.
4. Open a scene from `Assets/`.
5. Press Play in the Unity Editor.

## Notes

This project is focused on recreating a simple 2D arcade gameplay loop in Unity.
